Custom Nintendo 3DS Modding & Streaming

About project:

I desgined and installed a fully custom Pokémon-themed UI for a modded New 3DS, complete with original backgrounds, custom icon layout, and a background track ripped straight from Pokémon Fire Red/Leaf Green. The project involved deep interaction with legacy hardware, system-level modding tools, and network-based file transfers between my PC and 3DS — including recovery from a softbrick caused by a corrupted theme audio file.

- Designed and deployed a fully custom Nintendo 3DS theme using a graphics editor, YATA+, Anemone3DS, and BCSTM audio conversion, including graphic and audio asset integration.

- Configured FTP and file transfer pipelines between PC and 3DS via ftpd; resolved permission and format issues during installation.

- Enabled real-time 3DS-to-PC streaming using NTR CFW and Snickerstream, optimizing latency and video quality for gameplay footage.

- Diagnosed and fixed boot issues caused by theme bugs (e.g. mono audio incompatibility), troubleshooting through GodMode9 file system.

Mobile Application (Android) (Team)

About project:

Application developed in collaboration with 5 other students for a software engineering course at the University of California San Diego. The process was based on the Agile methodology and included a planning phase, design, development and testing. The team had stand up meetings at the beginning or ending of a class and also met once or twice a week for pair programming and other tasks. Planning phase including reading the requirements, activities like planning poker, creating user stories, test cases and tasks divided by phases. My tasks included testing and development.

Description: Allows users to input the coordinates of places and share their locations with other users so that the locations are displayed on a user interface that resembles a compass (circular view). The location updates in real time and users can set custom labels on the interface.
